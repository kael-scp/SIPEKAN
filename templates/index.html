Sistem Informasi Prediksi Kecepatan Angin (SIPEKAN)


Produk Data Science ini ditujukan untuk membantu BMKG Kota Batam memperkirakan kecepatan angin di Kota Batam dengan memanfaatkan model perkiraan kecepatan angin univariat terlatih (model perkiraan deret waktu) untuk memprediksi kecepatan angin. Model Neural Network Long Short Term Memory (LSTM) telah dilatih untuk mencapai hal ini. Hanya dengan memberikan 80 data per tiga jam nilai atribut prediktor selama 10 hari terakhir sebagai input ke sistem ini, kecepatan angin untuk 3 hari mendatang dapat diramalkan.
